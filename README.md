Quick start

In the project directory, you can run:

npm i

npm start

Runs the app in the development mode.
Open http://localhost:3000 to view in the browser.

The page will reload if you make edits.

The tech stack is:

- HTML5
- CSS3
- Flexbox
- Sass(scss)
- BEM methodology
- JS ES6
- React JS
- Redux
- Redux-thunk
- REST API
- ESLint
- npm
- prop-types
